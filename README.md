# CS2520 – Python for Programmers

A personal repo for coursework completed in **CS2520: Python for Programmers**.

## 🐍 Course Overview

An introduction to Python programming, covering core concepts such as syntax, semantics,
core data structures, control flow, functional and object-oriented programming paradigms,
input/output handling, and exception management.

View [Python's 3.14.3 documentation](https://docs.python.org/3/).
